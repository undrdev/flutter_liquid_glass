## 0.1.0

* Initial release of Flutter Liquid Glass package
* Implemented `LiquidGlassContainer` - core glassmorphic container with blur, tint, and edge shine
* Implemented `EdgeShinePainter` - custom painter for dynamic edge highlighting
* Implemented `LiquidGlassAnimationController` - animation utilities with spring physics
* Implemented `LiquidGlassCard` - glassmorphic card with touch interactions
* Implemented `LiquidGlassButton` - interactive button with press feedback
* Implemented `LiquidGlassSheet` - bottom sheet with glass effects
* Implemented `LiquidGlassAppBar` - translucent app bar
* Implemented `LiquidGlassNavigationBar` - bottom navigation with blur
* Implemented `LiquidGlassSnackBar` - floating snackbar with animations
* Implemented `LiquidGlassTheme` - theme extension for consistent styling
* Added comprehensive widget tests
* Added golden tests for visual regression
* Created demo application showcasing all features
* Platform support: iOS, Android, Web, macOS, Windows, Linux

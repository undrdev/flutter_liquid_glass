# Flutter Liquid Glass Example

This is a demo application showcasing the Flutter Liquid Glass package features.

## Running the Example

To run this example:

1. Navigate to the example directory:
```bash
cd example
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Features Demonstrated

- **Glass Containers**: Basic glassmorphic effects with blur and tint
- **Interactive Cards**: Touch-responsive glass cards with animations
- **Buttons**: Glass effect buttons with press feedback
- **Bottom Sheets**: Modal sheets with glass styling
- **Snackbars**: Glassmorphic notifications

## Try It Out

- Tap the cards to see touch animations
- Press the buttons to see the glass effects
- Open the bottom sheet to view the glass modal
- Show the snackbar for a glass notification
